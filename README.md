# ДачаПро (DachaPro)

Мобильное веб-приложение для садоводов и огородников. Помогает вести календарь дачных работ, получать ИИ-советы по дате, покупать партнёрскую продукцию «Органик Микс» на витрине и читать статьи.

## Стек

- React 18 + TypeScript + Vite
- React Router 6
- localStorage для пользовательских данных
- **Groq API** (модель `llama-3.1-8b-instant`) — бесплатный, ~30 запросов/мин
- CSS Modules, mobile-first (max-width 480px)

## Разделы

- **Главное** — приветствие, текущая дата, лунная фаза, задачи на сегодня, превью продуктов
- **Календарь** — сетка месяца, записи пользователя, кнопка «Совет от ИИ», рекомендации Органик Микс на дату
- **ИИ Агроном** — чат с Groq (бесплатно)
- **Витрина** — каталог продуктов партнёра «Органик Микс» (organic-mix.ru)
- **Статьи** — обучающие материалы

## Запуск

```bash
npm install
npm run dev   # http://localhost:5173
```

## ИИ-чат (страница «ИИ Агроном»)

Используется **Groq** — единственный провайдер. Бесплатный, быстрый (~0.5–2 сек на ответ), лимит ~30 запросов/мин.

1. Зарегистрируйтесь на https://console.groq.com/ (нужна только почта).
2. Создайте API-ключ.
3. Откройте файл `.env` в корне проекта и замените `gsk-REPLACE_ME` на свой ключ:
   ```
   VITE_GROQ_API_KEY=gsk-ваш-реальный-ключ
   ```
4. Перезапустите `npm run dev`.

> **Примечание:** ключ публичный и коммитится в репозиторий намеренно — чтобы обычные пользователи приложения не возились с настройками. Перед публикацией замените на свой.
